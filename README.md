##KURALLAR

-Oyun açılınca random bir şekilde ördekler yukarıdan aşşağıya düşer. 

-Vurulan ve kaçırılan ördekler soldaki menüde sayılır.

-Değiştirilebilir ama bu kodda toplam 50 ördek düşer.
